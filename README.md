# Networking-Foundations
Structured hands-on networking fundamentals practice, covering OSI model, IP addressing, subnetting, routing basics, and troubleshooting using Cisco Packet Tracer.
This repository documents my structured progression through core networking fundamentals.

The focus areas include:
OSI Model (Layer 1–7 understanding)
IP Addressing & Subnetting
Static configuration
Same-subnet vs multi-subnet communication
Basic routing concepts
ICMP testing and troubleshooting
Hands-on labs using Cisco Packet Tracer

Each lab includes:
Topology screenshots
Configuration details
Failure testing
Written analysis of observed behavior
